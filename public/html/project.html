Node.js / Express를 이용한 게시판
Explanation
Node.js / Express를 이용하여 게시판과 갤러리를 제작하였습니다. MySQL문법을 사용하여 Sequel Pro 데이터베이스에 데이터를 저장하도록 구현하였습니다. 마지막으로 아마존 웹 서비스(AWS)를 사용하여 웹 애플리케이션을 호스팅하였습니다.
* 회원가입도 가능하며, demouser로 로그인해 글작성도 가능하도록 구현하였습니다.
Used
Node.js, MySQL, Sequel Pro, Nuxt, Vuetify, JS, Pug, SCSS, fs-extra, Bootstrap, aws, masonry, tinyMCE, dotenv, 반응형
Link
http://15.165.194.171:3000/board
